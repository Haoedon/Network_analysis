**Настройка брандмауэра Windows с использованием программы Netsh**

1. Пропишем команду netsh.
2. Просмотрим список контекстов.

![Лаб9](<Основы%20сетевых%20технологий/Attachments/Лаб9%205.png>)

1. Выйдем из контекста.
2. Просмотр текущей конфигурации брандмауэра.

![Лаб9](<Основы%20сетевых%20технологий/Attachments/Лаб9%206.png>)

1. Просмотр правил для ENSP.

![Лаб9](<Основы%20сетевых%20технологий/Attachments/Лаб9%207.png>)

1. Отключим брандмауэр.

![Лаб9](<Основы%20сетевых%20технологий/Attachments/Лаб9%208.png>)

1. Включим обратно.

![Лаб9](<Основы%20сетевых%20технологий/Attachments/Лаб9%209.png>)

1. Сохраним настройки брандмауэра в файл. Для этого введём команду: netsh advfirewall export “C:\temp\WFconfiguration.wfw”.

![Лаб9](<Основы%20сетевых%20технологий/Attachments/Лаб9%2010.png>)

1. Восстановим настройки брандмауэра по умолчанию. Для этого введём команду: netsh advfirewall reset.

![Лаб9](<Основы%20сетевых%20технологий/Attachments/Лаб9%2011.png>)

1. Введём команду: netsh advfirewall firewall show rule name=eNSP.

Эта команда показывает правила брандмауэра Windows, специфичные для программы eNSP.

Rule Name: eNSP Tutorial

Enabled: Yes

Direction: In

Profiles: Domain, Private, Public

Grouping: eNSP

LocalIP: Any

RemoteIP: Any

Protocol: TCP

LocalPort: 12000

RemotePort: Any

Edge traversal: No

Action: Allow

Rule Name: eNSP Simulation

Enabled: Yes

Direction: Out

Profiles: Domain, Private, Public

Grouping: eNSP

LocalIP: Any

RemoteIP: Any

Protocol: UDP

LocalPort: Any

RemotePort: 67

Action: Allow

1. Откроем брандмауэр для программы eNSP. Для этого введём команду: netsh advfirewall firewall add rule name=“eNSP” dir=in action=allow pro gram=“C:\Program Files\Huawei\eNSP\eNSP_Client.exe” enable=yes.

![Лаб9](<Основы%20сетевых%20технологий/Attachments/Лаб9%2012.png>)

1. Введём команду netsh advfirewall firewall show rule name=eNSP. Отображаются правила брандмауэра ENSP.

![Лаб9](<Основы%20сетевых%20технологий/Attachments/Лаб9%2013.png>)

1. Создадим правило открытия порта 80. Для этого введём команду: netsh advfirewall firewall add rule name=“Open Port 80” dir=in action=allow proto col=TCP localport=80.

![Лаб9](<Основы%20сетевых%20технологий/Attachments/Лаб9%2014.png>)

1. Удалим правило для порта 80. Для этого введём команду: netsh advfirewall firewall delete rule name=“Open Port 80” protocol= TCP localport=80.

![Лаб9](<Основы%20сетевых%20технологий/Attachments/Лаб9%2015.png>)

1. Удалим правило для программы eNSP. Для этого введём команду: netsh advfirewall firewall delete rule name=“брандмауэр” program=“C:\Program Files\Huawei\eNSP\eNSP_Client.exe”

![Лаб9](<Основы%20сетевых%20технологий/Attachments/Лаб9%2016.png>)

1. Восстановим настройки из файла. Для этого введём команду: netsh advfirewall import “C:\temp\WFconfiguration.wfw”.

![Лаб9](<Основы%20сетевых%20технологий/Attachments/Лаб9%2017.png>)
